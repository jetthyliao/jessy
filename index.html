CSMPlug provides a worst-case prediction for hydrate plug dissociation time by three methods: one-sided depressurization, two-sided depressurization, and by electrical heating. The program also contains a safety simulator.


The plug dissociation programs are written in Fortran Code. The safety simulator is written in C#. All sections of the program are interfaced with C#.


Each section of the program will automatically display a summary of the important results from the calculation. The program also writes additional data to separate files.
